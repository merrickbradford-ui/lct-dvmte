# 附近医院路线导航网页

这是一个帮助用户查找附近医院并提供路线导航的网页应用。

## 功能特性

- 📍 获取用户当前位置
- 🏥 显示附近的医院列表
- 🗺️ 在地图上显示医院位置和标记
- 🚗 提供到指定医院的路线信息
- 🔍 支持医院名称搜索
- 📱 响应式设计，支持移动设备

## 技术栈

- HTML5
- CSS3 (包含响应式设计)
- JavaScript
- 百度地图API (WebGL版本)

## 使用说明

1. **获取百度地图API密钥**：
   - 访问 [百度地图开放平台](https://lbsyun.baidu.com/)
   - 注册账号并创建应用
   - 获取AK (Access Key)

2. **配置API密钥**：
   - 打开 `index.html` 文件
   - 将第8行中的 `您的百度地图AK` 替换为您的实际AK

3. **本地运行**：
   - 直接在浏览器中打开 `index.html` 文件
   - 或使用本地服务器（推荐）：
     ```bash
     # 使用Python启动本地服务器
     python -m http.server 8000
     # 然后访问 http://localhost:8000
     ```

## 注意事项

- 由于使用了地理位置API，需要在HTTPS环境下或本地文件系统中运行
- 当前版本使用模拟数据，实际部署时需要集成真实的医院数据API
- 百度地图API需要有效的AK才能正常工作

## 自定义

- 可以修改 `mockHospitals` 数组中的数据来添加更多医院
- 样式可以在 `styles.css` 中进行调整
- 功能扩展可以在 `script.js` 中进行开发
