# 徒步路线探索网站

这是一个展示附近徒步路线的响应式网页项目，具有现代化的UI设计和完整的功能。

## 功能特性

- **响应式设计**：完美适配移动设备、平板和桌面浏览器
- **地图集成**：静态地图展示区域（可扩展为真实地图API）
- **徒步路线列表**：展示路线名称、难度、距离、预计时间、评分等信息
- **搜索和筛选**：按关键词、地区、难度进行筛选
- **现代化UI**：绿色主题，卡片式布局，悬停效果
- **评分系统**：显示用户评分（星级）
- **重置功能**：一键清除所有筛选条件

## 运行说明

### 方法1：直接打开（推荐）
1. 在文件资源管理器中找到 `index.html` 文件
2. 双击打开，或右键选择"在浏览器中打开"

### 方法2：本地服务器（开发推荐）
如果你有Python环境，可以使用以下命令启动本地服务器：

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然后在浏览器中访问 `http://localhost:8000`

### 方法3：其他本地服务器工具
- 使用 VS Code 的 Live Server 插件
- 使用 Node.js 的 `http-server` 包
- 使用任何静态文件服务器工具

## 数据说明

项目包含8条精选的中国著名徒步路线数据，包括：
- 黄山徒步路线（华东）
- 张家界天门山玻璃栈道（华中）
- 稻城亚丁徒步路线（西南）
- 泰山十八盘（华北）
- 长白山天池环线（东北）
- 梅里雪山雨崩徒步（西南）
- 华山长空栈道（西北）
- 桂林漓江徒步（华南）

每条路线都包含详细的描述、图片、难度等级、距离、预计时间和用户评分。
