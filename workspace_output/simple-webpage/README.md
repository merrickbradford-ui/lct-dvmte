# 简单网页示例

这是一个基础的静态网页项目，展示了HTML、CSS和JavaScript的基本用法。

## 文件结构

- `index.html` - 网页的主要HTML结构
- `styles.css` - 网页的样式表，包含响应式设计
- `script.js` - 网页的交互功能脚本

## 功能特点

1. **响应式设计** - 在桌面、平板和手机等不同设备上都能良好显示
2. **现代CSS样式** - 使用渐变背景、阴影效果和流畅的过渡动画
3. **交互功能**：
   - 点击"点击改变背景颜色"按钮可以循环切换不同的背景色
   - 点击"点击显示消息"按钮会随机显示一条欢迎消息
   - 页面具有视觉反馈效果，提升用户体验

## 如何使用

1. 将整个`simple-webpage`文件夹下载到本地
2. 在浏览器中直接打开`index.html`文件即可查看和使用
3. 无需服务器，纯静态文件即可运行

## 自定义修改

- 要修改内容：编辑`index.html`文件
- 要修改样式：编辑`styles.css`文件  
- 要修改交互：编辑`script.js`文件

这个示例适合初学者学习网页开发的基础知识，也可以作为更复杂项目的起点。
