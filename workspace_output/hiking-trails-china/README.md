# 全国徒步路线探索网站

一个展示中国范围内徒步路线的静态网页，包含地图展示和搜索功能。

## 功能特点

- **地图展示**：使用 Leaflet 和 OpenStreetMap 展示徒步路线位置
- **搜索功能**：支持按城市或地点名称搜索徒步路线
- **当前位置**：可使用浏览器地理位置 API 定位并显示附近路线
- **响应式设计**：适配桌面和移动设备

## 技术栈

- HTML5 / CSS3 / JavaScript (ES6+)
- Leaflet.js 地图库
- OpenStreetMap 地图瓦片

## 使用说明

1. 直接在浏览器中打开 `index.html` 文件
2. 在搜索框中输入城市名称（如“杭州”）并点击“搜索附近路线”
3. 或点击“使用我的当前位置”按钮获取当前位置附近的路线

> 注意：由于使用了模拟数据，实际搜索结果仅限于预设的几条路线。在生产环境中，应替换为真实的徒步路线 API。

## 数据说明

当前项目使用模拟数据（mock data）展示功能。真实项目中可集成以下数据源：
- OpenStreetMap 的 hiking trails 数据
- 高德地图/百度地图 API（需注意中国地理数据合规要求）
- 自建徒步路线数据库

## 浏览器兼容性

- 支持现代浏览器（Chrome, Firefox, Safari, Edge）
- 地理位置功能需要 HTTPS 环境（本地文件可能受限）
