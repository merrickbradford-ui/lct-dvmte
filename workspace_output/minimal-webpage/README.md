# 最小网页示例

这是一个展示最小可行网页的示例项目，包含基本的HTML、CSS和JavaScript文件。

## 文件结构

- `index.html` - 主HTML文件，包含页面结构
- `styles.css` - CSS样式文件，定义页面外观
- `script.js` - JavaScript文件，添加交互功能

## 如何运行

1. 将所有文件保存到同一目录
2. 在浏览器中打开 `index.html` 文件
3. 点击页面上的按钮查看交互效果

## 特点

- 代码简洁，易于理解
- 包含基本的网页三要素：结构(HTML)、样式(CSS)、行为(JavaScript)
- 响应式设计，适配不同屏幕尺寸
- 无需服务器，可直接在浏览器中运行
